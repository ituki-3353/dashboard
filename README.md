# Dashboard Project

パーソナルな情報を一元管理するためのウェブダッシュボードシステムです。

## 機能概要

- **時計表示**: リアルタイムで時刻を表示。
- **検索機能**: ウェブ検索ボックスとクイックリンク。
- **ウェザー**: 現在地の天気情報を表示。
- **ニュース**: 最新ニュースのリスト表示と手動更新機能。
- **システム情報**: 実行環境のシステムステータスを表示。
- **メモ帳**: ブラウザ上で利用可能な簡易メモ機能。
- **ローディング画面**: システム起動時のステータスログとアニメーション表示。

## ファイル構成

- `index.html`: メインのレイアウト構造。
- `style.css`: デザイン・スタイル定義。
- `config.js`: APIキーや各種設定値の管理。
- `main.js`: 各種ウィジェットのロジック、データ取得処理。

## UIレイアウト

- **Sidebar (Left)**: 天気、ニュース。
- **Main Content**: 時計、中央メッセージ、検索セクション。
- **Sidebar (Right)**: システム情報、メモ。

## 技術スタック

- HTML5 / CSS3
- JavaScript (Vanilla JS)
- Font/Icons: Icons8

## セットアップ

1. `config.js` で必要なAPIキーなどを設定します。
2. `index.html` をブラウザで開くことで実行可能です。
